Projects

Here is a list of projects I have worked on, either by myself or with other people. Feel free to contact me if you have any questions or comments!

  • Paparazzi: A social game developed using Ruby on Rails with other interns at Viget. In Paparazzi, a player can win by taking pictures of other participants without getting caught themselves.

  • Haxxor Clone: A clone of the popular news website Hacker News made using Ruby on Rails. This application was also developed while I was interning at Viget.

  • WatchMe: An application made for the Pebble Watch for user safety. A user can open the application and call an emergency contact with an autonomous message containing the user's location.
